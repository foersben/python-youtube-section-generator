# Core Services

Overview of core business logic modules in `src/core/`:

- `transcript` — extraction from YouTube
- `services` — SectionGenerationService, TranscriptRefinementService, translation
- `retrieval` — RAG indexing and section generation
- `llm` — local and Gemini adapters

Refer to inline module docstrings in `src/core/` for more details and code examples.
